## 🧪 Simple Troubleshooting

If Argus Automation does not start:
- open the file again
- right-click and choose **Run as administrator**
- check that Windows did not block the app
- make sure the file finished downloading

If the agent does not click the right place:
- bring the app window to the front
- make sure the screen scale is set to a normal value
- keep the window size the same during the task
- avoid moving windows while the agent runs

If typing looks wrong:
- check your keyboard layout
- close other apps that may be taking input
- restart the app and try again

If the screen goes dark or locks:
- keep the PC awake
- disable sleep during long tasks
- do not lock the session while automation is active
